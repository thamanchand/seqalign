============
Requirements
============

Seq-alignment tools can be used in varied systems base on whether all intermediary applications
are available or not like in Windows, Mac, Linux and UNIX. But, it has been tested on the Windows
working under a beta version.

System
------

1. Windows

2. Mac

3. Linux

Applications
------------

1. Python 2.5 or above

2. Biopython 1.5 above (Must be compatible with python version)

3. Clustalw2

4. PHYLIP

Servers
-------

Apache web server 2.* above

Browser
-------

1. Mozilla Firefox

2. Internet explorer

3. Safari

4. Netscape
